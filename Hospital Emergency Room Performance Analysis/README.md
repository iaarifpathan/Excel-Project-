# 🏥 Hospital Emergency Room Performance Analysis

## 📌 Project Overview
This project focuses on analyzing hospital emergency room data using Excel to improve operational efficiency and provide actionable insights. The report helps stakeholders monitor patient trends, wait times, and service quality to make better decisions.

---

## 🎯 Objective
To build an interactive Excel-based analytical report that:
- Tracks patient flow and ER performance
- Identifies delays and inefficiencies
- Improves decision-making for hospital management

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Power Query (Data Cleaning & Transformation)
- Power Pivot (Data Modeling)
- DAX (Calculated Columns & Measures)
- Pivot Tables & Charts

---

## 📊 Key KPIs
- **Number of Patients** → Daily patient count with trend analysis  
- **Average Wait Time** → Time patients wait before treatment  
- **Patient Satisfaction Score** → Service quality indicator  

---

## 📈 Analysis & Visualizations
- Patient Admission Status (Admitted vs Not Admitted)
- Patient Age Distribution
- Timeliness (Patients attended within 30 minutes)
- Gender Analysis
- Department Referral Analysis

---

## 🔄 Project Workflow

1. Business Requirement Gathering  
2. Data Understanding  
3. Data Import using Power Query  
4. Data Cleaning & Quality Check  
5. Calendar Table Creation (Power Query)  
6. Data Modeling using Power Pivot  
7. DAX Calculations  
8. Pivot Tables & Chart Creation  
9. Report Development  
10. Insights Generation  

---

## 🧹 Data Preparation

- Merged **First Name + Last Name** → Created Patient Name  
- Converted **Boolean Admission Flag**:
  - TRUE → Admitted  
  - FALSE → Not Admitted  
- Handled missing values and ensured correct data types  

---

## 📅 Calendar Table

A calendar table was created using Power Query to enable time-based analysis.

### Why it is used:
- Analyze data by Month, Year, Trends  
- Handle missing dates  
- Enable time-based calculations  
- Build relationships with main dataset  

**Relationship Created:**
- ER Data (Admission Date) ↔ Calendar Table (Date)
