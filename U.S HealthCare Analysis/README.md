## 🔍 Project Objectives
- Analyze the prevalence of medical conditions across gender, age groups, and blood types.  
- Study billing patterns by admission type and insurance providers.  
- Create an **interactive Excel dashboard** to visualize key insights.  
- Provide conclusions that support **healthcare strategy and resource allocation**.  

---

## 🛠️ Workflow

### 1. **Data Cleaning**
During the cleaning phase, I fixed inconsistencies and errors in multiple columns:  
- **Age** → Corrected invalid entries and ensured numeric format.  
- **Gender** → Standardized inconsistent labels (e.g., “M” → “Male”).  
- **Blood Type** → Removed duplicates, corrected spelling errors, standardized values.  
- **Date of Admission** → Fixed inconsistent date formats into `dd/mm/yyyy`.  
- **Billing Amount** → Corrected typing errors, replaced missing values with median (20280.42).  
- **Admission Type** → Fixed misspellings (e.g., “Emer” → “Emergency”).  

✅ Final Output → A clean, consistent dataset ready for analysis.

---

### 2. **Data Transformation**
- Created **Age Buckets (3 Categories)** → *Young (<35), Middle (<60), Senior (≥60)*.  
- Created **Detailed Age Groups** → *0–18, 19–35, 36–50, 51–65, 66+*.  

✅ Final Output → Enhanced demographic segmentation for deeper insights.  

---

### 3. **Dashboard Creation**
Using Excel features like **Pivot Tables, Pivot Charts, Conditional Formatting, and Slicers**, I built an interactive dashboard with:  
- Distribution of medical conditions by gender, blood type, and age group.  
- Admission type & billing analysis.  
- Insurance provider analysis.  
- Key insights summarized below each chart.  

---

## ✨ Key Insights
1. **Hypertension** is the most prevalent condition (**22% of total cases**), with higher risk among **males (61%)**.  
2. **Obesity and Diabetes** are more common in females, indicating possible lifestyle/metabolic factors.  
3. **Emergency admissions** drive the highest revenue ($87M) and average billing ($24,279).  
4. **Cigna & Aetna** show the highest billing amounts, while **Medicare & UnitedHealthcare** have lower average costs.  
5. **Age groups** reveal that arthritis is highest in seniors, while asthma is more common in younger adults.
